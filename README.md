# Python商品推荐系统 python1262

本项目是一款基于协同过滤算法的商品推荐系统，集成了在线购物系统的基本功能。该系统不仅能够为用户提供个性化的商品推荐，同时支持用户进行登录、搜索、浏览商品、评论评分以及加入购物车等操作。

## 技术栈

- **后端**: Python + Django 框架
- **数据库**: MySQL
- **前端**: HTML + CSS + JavaScript

## 功能模块

### 用户端

- 登录与注册
- 商品搜索
- 商品浏览
- 评论与评分
- 购物车管理
- 商品购买

### 管理员端

- 商品增删改查
- 用户管理
- 订单管理
- 评论管理

## 系统角色

- **普通用户**: 使用推荐系统进行商品浏览和购买
- **管理员**: 系统维护和内容管理

## 运行环境

- Python 3.x
- Django 2.x
- MySQL 5.x
- 兼容主流浏览器

## 部署步骤

1. 环境准备：安装 Python、MySQL 和相关依赖库
2. 数据库配置：设置 MySQL 数据库
3. 代码部署：部署后端 Django 代码
4. 前端部署：部署 HTML、CSS 和 JavaScript 文件
5. 系统启动：运行 Django 应用
